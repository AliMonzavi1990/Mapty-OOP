# Mapty

**A JavaScript app to track running and cycling workouts on a map.**  

---

## Features
- Add running and cycling workouts with distance, duration, pace/speed, and elevation.  
- Display workouts as markers on a Leaflet map.  
- View workout details in a list.  
- Delete workouts (double-click).  
- Data persists in **LocalStorage**.  

---

## Technologies
- JavaScript (ES6+)  
- Leaflet.js (for interactive maps)  
- HTML / CSS  

---

## Demo
[Live Demo](https://maptytracking.netlify.app/)

---

## How to Use
1. Open the app in your browser.  
2. Click on the map to add a new workout.  
3. Fill in the form with distance, duration, and other details.  
4. Your workout appears on the map and in the list.  
5. Double-click a workout to delete it.  

---

## Notes
- This project uses browser **LocalStorage** to save workouts.  
- Works best on modern browsers with geolocation support.
